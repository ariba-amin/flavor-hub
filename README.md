# Flavor Hub

Flavor Hub is a responsive online food ordering website built using HTML, CSS, and JavaScript.

## Features
- Home page with featured food items
- Popular dishes section with Add to Cart functionality
- Menu page displaying food items and prices
- Shopping cart system
- Order summary and bill calculation
- Checkout form for customer details and payment method
- About page with service information
- Contact page with contact form
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
